# Учебный проект "Автоматизация тестирования API сервиса Яндекс.Самокат"

## Описание проекта
Реализованы автотесты API учебного сервиса Яндекс.Самокат.

Команда для отображения отчета Allure - mvn allure:serve


## Описание классов с тестами
* CourierRegistrationTest - проверка регистрации курьера
* CourierLoginTest - проверка логина курьера
* CourierDeleteTest - проверка удаления курьера
* OrderCreation - проверка создания заказа
* OrderList - проверка получения списка заказов
* OrderIdReceivingTest - проверка получения заказа по его номеру
* AcceptOrderTest - проверка принятия заказа

## Вспомогательные классы
* CommonTest - общий настроечный класс для выполнения перед и после всех тестов
* RequestApi - API запросов
* Пакет order - создание объекта заказа 
* Пакет courier - создание объекта курьера
* Пакет actions - вспомогательные методы
* CustomException - собственные исключения
* RequestErrors - текст ошибок API
* Urls - адреса API
